ring.geometry

The base idea of the app is to show the design potential of for in for loops in codes for design compositions. Therefore, a simple ring design provides the base geometry. The ring was translated on the x and y axis with a sin(x) function that creates the waved appearance. The multiplication with a factor, allows the user to change the amplitude and discover different configurations by only a small change of the parameters. The code is structured in three main definitions: main(), generate_ring() and update(). In the generate function() with a nested loop inside a loop, the ring geometry is created and translated in three dimensions. Later a update function() checks if the lenght of rings is not zero, and regenerates, when the geom_params values are not equal to the assigned variables.
